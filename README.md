# Jolayemi Nurudeen Olarewaju - Dynamic Portfolio

Welcome to my professional portfolio source code! 🚀 This repository contains a fully bespoke, responsive, and dynamic portfolio website designed to showcase my expertise in software development, AI automation, and technical problem-solving.

## 🌟 Modern Architectural Features

Unlike standard static templates, this portfolio is engineered with dynamic, serverless architectures to automate content delivery:

- **Live GitHub API Integration:** The "Projects" section dynamically fetches and displays my top 6 public repositories straight from the GitHub REST API, complete with descriptions and interactive buttons.
- **Serverless Admin CMS (Firebase):** A hidden `/admin.html` dashboard empowered by Google Firebase Authentication and Firestore. This allows me to securely log in and edit text content (like "About Me" and "Skills") instantly, without touching the code.
- **Dynamic CV Synchronization:** The "View CV" feature uses a live Google Drive embed modal. Whenever I update my resume on Google Drive, the website automatically syncs and displays the latest version.
- **API-Powered Contact Form:** Uses [FormSubmit.co](https://formsubmit.co) asynchronous AJAX technology. Messages sent through the platform bypass the need for a PHP backend, securely delivering straight to my inbox without page reloads.
- **Responsive Aesthetics:** Built with Bootstrap 5, featuring smooth micro-interactions, dark-mode elements, and flawless mobile responsiveness.

## 🛠️ Stack & Technologies

- **Frontend:** HTML5, modern vanilla CSS3, Vanilla JavaScript (ES6+), Bootstrap 5.
- **Backend/Database:** Serverless APIs, Google Firebase (Authentication & Firestore).
- **Integrations:** GitHub REST API, Google Drive Embed API, FormSubmit AJAX API.

## 🚀 Live Demo & Hosting

Since the entire application runs strictly on frontend technologies and serverless APIs, it is completely decoupled from traditional web servers. It perfectly executes on local servers (`Python http.server`, `Live Server`) and free static hosts like **GitHub Pages**, **Vercel**, or **Netlify**.
